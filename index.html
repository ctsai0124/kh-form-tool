⚙️ 「記錄＋寄信提醒」一次性設定（給紀錄用 Google 表單 + 提醒 Email）
第一步：建一份「甄選紀錄」表單（跟②分頁報到表單是不同份，這份是共用後端用的對照表）
1. 到 forms.google.com 建新表單，加入 4 個「簡答／段落」欄位： 　活動代碼、學校全銜、通知信箱、完整資料(段落)
2. 右上角「⋮」→「取得預先填入的連結」→ 四個欄位分別填入 ANNO_NO、SCHOOL、NOTIFY_EMAILS、FULLDATA 這四個文字（照打，供本工具辨識用；活動代碼欄位填 ANNO_NO 沒關係，這是內部代號）
3. 取得連結後複製，貼在下方「紀錄表單預填連結範本」
4. 這份表單連結的試算表，之後也要交給 Apps Script 讀取（見 README 的 Apps Script 設定章節）

尚未產生表單 — 請先於左側上傳並解析簡章。

② 產生報到 QR code

活動代碼不用讓考生填。本工具會把①分頁產生的活動代碼，直接「鎖」進 QR code 連結裡 （用 Google 表單的「預先填寫」功能），考生掃到的表單裡活動代碼那格已經固定好、看不到也改不了， 只需要填姓名／身分證後4碼／報考科別，不會有多場甄選、多校混在一起或打錯字的問題。
設定「範本連結」這個步驟只需要做**一次**，之後每場甄選只要換活動代碼重新產生 QR 就好。
還沒設定「表單預填連結範本」？點此看設定教學（整個系統只需做一次，所有學校共用這一份）
第一步：建立表單（如果還沒建過）
1. 到 forms.google.com 建立新表單，標題例如「甄選現場報到」
2. 加入以下 4 個「簡答」欄位：① 活動代碼　② 姓名　③ 身分證後4碼　④ 報考科別
3. 右上角「回覆」分頁 → 點綠色 Sheets 圖示 → 建立連結的 Google 試算表

第二步：取得「預先填入的連結」範本（重點步驟）
4. 在表單編輯畫面右上角「⋮」更多選項 → 點「取得預先填入的連結」
5. 畫面會列出表單所有欄位，只在「活動代碼」這一格填入任意文字（例如 TEMP），其餘欄位全部留空
6. 按「取得連結」→ 複製產生的網址 → 貼到上方「表單預填連結範本」欄位（瀏覽器會記住，之後不用再貼）

這份表單、試算表全系統只建一次，所有學校、所有場次都共用這一份，靠「活動代碼」分辨彼此， 不會混淆。之後在③「即時報到名單」分頁，還需要把試算表「發布到網路」為 CSV， 才能讓本工具讀到即時資料；這份試算表也是 Apps Script 寄送通知信要監聽的對象（見 README）。

③ 即時報到名單

把 Google 試算表「發布到網路」產生的 CSV 網址貼在下面（試算表選單：檔案 → 共用 → 發布到網路 → 格式選 CSV）。 本工具只會讀取，不會寫入你的試算表。名單會依「活動代碼」篩選，並依報到時間先後，在各科別內自動編上准考證號碼。

④ 甄選成績表

依報到名單自動產生（也可以在③分頁還沒設定好之前，直接在這裡手動增加考生列來測試格式）。 委員分數與平均為即時計算，「錄取資格」欄位請人事人員自行填寫判斷。
